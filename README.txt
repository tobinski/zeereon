README:
Zeereon ist ein Onlinezeiterfassung mit integrierter Auswertung und einem Rechnungsexport.

Install:
Zeereon verfügt über ein Installationsscript.
1.) Die Daten auf einen Apache Server kopieren
2.) Die URL des Servers aufrufen und den Aneitungen folgen
